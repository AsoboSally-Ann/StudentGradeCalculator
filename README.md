# StudentGradeCalculator
A student grade calculator built in Dart with Flutter desktop GUI and console mode  featuring data classes, nullable types, and functional programming.


## 🎓 Student Grade Calculator

A dual-mode Student Grade Calculator built in Dart and Flutter, developed across two milestones as part of a Kotlin/Dart programming course.

### Features
- Object-oriented design with Student and GradeEntry data classes
- Nullable inputs with Elvis (??) and safe-call (?.) operators
- Flutter desktop GUI with student roster, grade table, and class summary
- Pure Dart console mode with interactive menu
- Validation functions on data classes
- Functional programming: map, filter, forEach on collections
- Custom higher-order functions that accept lambdas as parameters
- Immutable data updates using copyWith pattern

 Modes
- **GUI** — Flutter desktop app (Windows/macOS/Linux) with Material 3 design
- **Console** — Pure Dart terminal app, runs with just the Dart SDK

Tech Stack
- Language: Dart 3
- Framework: Flutter (desktop + web)
- IDE: VS Code

 **How to Run**
Console (no Flutter needed):

dart run console_main.dart

**Desktop GUI:**

flutter pub get
flutter run -d windows
